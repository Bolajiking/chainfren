# Chainfren Website — New Copy

**Document purpose:** Drop-in copy replacements for the existing chainfren.com structure. Nothing changes about the UI, layout, components, animations, or visual design. Only the words on the page.

**How to use this document:** Each section maps to an existing page and an existing UI element. The structure of this document mirrors the structure of the site. For each element, the existing copy is shown for reference, then the new copy follows. The developer or designer pastes the new copy into the same field where the old copy lives.

**Strategic frame governing every word:** Chainfren is the growth engine behind Africa's creators. Sabi is Africa's onchain broadcasting network. Together they end the era of rented attention.

## Voice and Tone — applied throughout

Three rules govern every word in this rewrite:

**Rule 1: No hedge language.** "We aim to," "we strive to," "we believe" — banned. We *do*. We *build*. We *publish*. The brand has earned declarative sentences.

**Rule 2: Contrast is the engine.** Every important sentence sets up a contrast. Rented vs. owned. Followers vs. fans. Reach vs. revenue. Platforms vs. infrastructure. The contrast is what makes the argument feel inevitable.

**Rule 3: Lagos voice, not San Francisco voice.** When two phrasings are equally clear, pick the one that sounds like it was written by someone who actually lives in Lagos. "We go to work" beats "we execute." "Stop renting your audience" beats "audience ownership."

## Summary — What This Rewrite Changes

**Brand line:** Every page now leads with "the growth engine behind Africa's creators" or a direct extension of it.

**Sabi positioning:** The /media page repositions from "media production services" to "Africa's onchain broadcasting network." The "Playbook" tile on the homepage rebrands as "Sabi." The newsletter footer rebrands as "Join Sabi."

**Star Factor visibility:** Star Factor appears prominently on the homepage and Sabi page (as "Sabi presents Star Factor") and optionally as a fourth product tile if the UI allows.

**Agency sharpening:** The agency page's six service tiles keep the same structure but the descriptions tighten significantly — outcome-language replaces buzzword-language.

**Contact form:** Dropdown options reorganized to match the actual product stack and the Sabi rebrand.

**What does not change:** The UI design. The page structure. The component layout. The animations. The visual identity. The number of tiles per page. The button positions. The navigation. The footer structure. The form layout. None of it. Only the words on top.

## Implementation Notes for Developer / Designer

The implementation is text-only. The recommended sequence:

**Phase 1 — Critical pages (homepage, /media):** ship the homepage hero, argument paragraph, services tile, "for creators" tile, and "Playbook → Sabi" rebrand on the homepage; ship the full /media page rewrite as Sabi's destination. These are the pages that carry the brand line publicly.

**Phase 2 — Supporting pages (/agency, /products):** ship the agency tile rewrites and product tile rewrites. The structure is unchanged so this is straight find-and-replace.

**Phase 3 — Functional pages (/contact):** update the contact form copy and dropdown options.

**Don't ship anything until all three phases are ready** — partial rollouts where the homepage talks about "Sabi" but /media still says "MEDIA" create user confusion. Either ship the whole rewrite or hold it.

If the TVinBio tile copy changes are too aggressive for the current free/paid positioning of the product, dial back to: *"Replace your link-in-bio with a media hub you own. Capture fans, sell directly, run your storefront from one page. Free to start."* — slightly shorter, equally directional, less commitment to specific feature claims.
